Koa-Middle
==========

Simple Koa middle example

Requirements
------------
Paddington requires the following to run:

* [Node.js](http://nodejs.org/) 8+
* [npm](https://www.npmjs.com/) (normally comes with Node.js)

Usage
-----

### How to Start

```sh
# Install Node.js components listed in ./package.json
npm i

# Compile and launch
npm start
```
